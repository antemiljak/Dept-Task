## Getting Started

To run this project locally, follow these steps:

### 1. Clone the Repository


```bash
git clone https://github.com/antemiljak/Dept-Task.git
```
### 2. Navigate to folder
```bash
cd Dept-Task
```
### 3. Install Dependencies
```bash
npm i
```
### 4. Start the Project Locally
```bash
npm run dev
```
